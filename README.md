# E-Commerce Angular

A modern e-commerce web application built with Angular 21. It features a component-based architecture using standalone components and Angular Signals for reactive state management. The project follows strict code quality standards enforced by ESLint, Prettier, and Conventional Commits.

## Commands

### Dev

| Command          | Description                           |
| ---------------- | ------------------------------------- |
| `npm start`      | Dev server at `http://localhost:4200` |
| `npm run build`  | Production build to `dist/`           |
| `npm test`       | Run unit tests with Vitest            |
| `npm run lint`   | Lint with ESLint                      |
| `npm run format` | Format with Prettier                  |

### Generate

| Command                          | Description                                     |
| -------------------------------- | ----------------------------------------------- |
| `ng generate component <name>`   | Component (`.ts`, `.html`, `.scss`, `.spec.ts`) |
| `ng generate service <name>`     | Injectable service                              |
| `ng generate directive <name>`   | Attribute directive                             |
| `ng generate pipe <name>`        | Transformation pipe                             |
| `ng generate guard <name>`       | Route guard                                     |
| `ng generate interceptor <name>` | HTTP interceptor                                |
| `ng generate interface <name>`   | TypeScript interface                            |
| `ng generate enum <name>`        | TypeScript enum                                 |
| `ng generate class <name>`       | Plain TypeScript class                          |
